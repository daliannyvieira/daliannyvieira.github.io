Redesign 2.0

novembro, 2021 — 2 mins de leitura

Do gatinho puto até a bandeira lgbtqia+ foram as minhas inspirações nessa versão nova.

Dark theme

Sem dúvidas o que mais me motivou a mexer em tudo de novo foi a necessidade de desenvolver a funcionalidade do tema escuro. Não que isso fosse tão importante assim, mas parecia um desafio divertido e me trouxe alguns ganhos estéticos e funcionais.

Para habilitar a troca de themes eu testei coisas legais como:

  • Context api: Para criar um theme provider
  • CSS Variables: Para armazenar as cores de cada theme (e que saudades, meus amigos)
  • CSS Animations: Para criar a animações no menu, novo footer, links para redes sociais, postits voadores para compartilhar postagens, e algumas outras pequenas espalhadas pelo site (outra saudade enorme)

Cores por toda parte

Uma das coisas que mais me incomodavam no layout antigo era a página da listagem de postagens, e a própria página de artigos que eram praticamente sem cor. Eu também recebi um feedback de que a leitura dos artigos estava muito prejudicada pelas letras pretas sobre o fundo branco.

Pois bem, eu peguei o balde de tinta e derramei sem dó, e agora eu tenho:

  • Uma nova paleta de cores que eu estou chamande de COLOR_RANGE e ela está por toda parte;
  • Uma nova página de listagens de posts com uma cor destaque para cada ano baseada na nova paleta;
  • Um menu de navegação colorido baseado na nova paleta
  • Alguns componentes escolhem uma cor aleatória a cada render baseada no color range, esse ainda vai mudar muito;

Nova home

A página inicial agora conta com animação muito divertida - que eu ainda preciso refatorar - que desenha pingos de tinta sobre página. Para fazer isso eu fiz o seguinte:

Toda vez que alguém clica eu adiciono um novo círculo colorido com uma cor aleatória baseada na minha paleta de cores e um tamanho aleatório baseado também em uma lista de tamanhos

Cada círculo é uma div renderizada usando styled components - o que não foi a escolha mais performática do mundo - mas para um desenho simples como eu pretendia habilitar fazer até que serviu bem e essa brincadeira me rendeu vários prints dos meus amigos!
